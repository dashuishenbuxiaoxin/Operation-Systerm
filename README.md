##数据结构：

///磁盘：
1.位示图-------整型数组
2.目录项-------FCB结构体
3.目录块-------DIR结构体
4.i节点块------INODE结构体
5.数据块-------字符数组	
6.磁盘---------DISK结构体

///内存：
1.用户打开文件表------openlist结构体
2.用户表--------------User结构体
3.路径表--------------Path结构体
